# Wetterabfrage-Tool 🌤️

Ein Python-CLI-Tool das das aktuelle Wetter für eine beliebige Stadt abruft.

## Funktionen
- Stadtsuche per Eingabe
- Automatische Koordinatenermittlung via Geocoding API
- Aktuelles Wetter (Temperatur, Wind, Zustand)

## Technologien
- Python 
- requests
- Open-Meteo API (kostenlos, kein API-Key nötig)

## Nutzung
```bash
python wetterabfrage.py
```

## Beispielausgabe
```
--- Wetter in Gevelsberg ---
Zustand:             Bewölkt
Temperatur:          8.4°C
Windgeschwindigkeit: 10.1 km/h
```
